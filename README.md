# University Admissions Bot

## Description
The University Admissions Bot is an AI-powered chatbot that helps students
understand university admission requirements by compressing long guidelines
into short, easy-to-understand responses.

## Features
- Admission eligibility checking
- Document requirement summaries
- Application deadline information
- Conversational interface

## Tech Stack
- Python
- Streamlit
- JSON-based data storage

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py

## Use Case
Helps students quickly understand university admission requirements without
reading lengthy documentation.
